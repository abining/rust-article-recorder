## 4. 接口文档汇总 (API Blueprint)

当你完成上述代码生成后，你的系统将暴露以下接口。为了方便你后续开发前端或测试，以下是接口清单：

### 认证接口 (Public)

| **接口**               | **方法** | **说明** | **备注**                 |
| -------------------- | ------ | ------ | ---------------------- |
| `/api/auth/register` | `POST` | 用户注册   | 需输入 username, password |
| `/api/auth/login`    | `POST` | 用户登录   | 返回 Token               |

### 文章管理接口 (Protected - 需要 JWT)

| **接口**              | **方法**   | **说明**     | **备注**     |
| ------------------- | -------- | ---------- | ---------- |
| `/api/articles`     | `GET`    | 获取当前用户所有文章 | 包含私密文章     |
| `/api/articles`     | `POST`   | 创建新文章      | 需定义唯一 slug |
| `/api/articles/:id` | `PUT`    | 修改内容/状态    | 只能操作自己的文章  |
| `/api/articles/:id` | `DELETE` | 删除文章       | 只能操作自己的文章  |

### 内容访问接口 (Global/Conditional)

| **接口**   | **方法** | **说明**       | **规则**                    |
| -------- | ------ | ------------ | ------------------------- |
| `/:slug` | `GET`  | **根据路由获取文章** | 公开文章直接看；私密文章仅作者带 Token 可看
