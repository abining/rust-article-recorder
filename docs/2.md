阶段 2：数据库建模与 JWT 鉴权 (Auth & DB)
文件名：prompt_02_auth.md

提示词内容： 基于第一阶段的项目，请实现用户系统和鉴权逻辑。

要求：

SQL 迁移： 创建 users 表。字段：id (Binary(16) 存储 UUID), username (唯一), password_hash。

认证模块：

使用 bcrypt 处理密码加密。

实现 /api/auth/register 和 /api/auth/login。

登录成功后返回有效期 24 小时的 JWT Token。

Axum 中间件： 实现一个 AuthMiddleware。它拦截请求头中的 Authorization: Bearer <token>，验证通过后将 user_id 注入到请求上下文中（Request Extension）。

请输出： src/models/user.rs、src/handlers/auth.rs 以及中间件的 Rust 实现代码。
